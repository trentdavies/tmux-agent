# Phase 2: Communication + Adoption

## What
Add `ta send` for sending text to agent panes and `ta adopt` for adopting pre-existing tmux sessions.

## Why
Enable programmatic interaction with agents (send prompts, broadcast messages) and onboard existing sessions into the naming convention.

## Key Features
- Send by pane index, agent type, or broadcast
- Adopt: rename pane titles to follow naming convention
- Dry-run mode for adoption
